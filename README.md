# Curriculum Visualization 

This project is a **responsive academic curriculum map** built using [Middleman](https://middlemanapp.com/).  
It visually displays a 4-year Computer Science and Engineering degree plan with interactive features, modern design, and responsive behavior across devices.

---

## Features

- **Highlights future, past prerequisites** - Highlights with different colors what classes to be taken as pre requisite for a particular class. Also, it highlights which classes to take in future for continuing C.S. track.
- **Responsive Grid Layout** – Adjusts from 4 columns (desktop) → 2 (tablet) → 1 (mobile)
- **Modern UI** – Clean, simple, and professional design
- **Interactive Hover Effects** for course sequences
- **Structured Middleman Architecture** for easy editing
